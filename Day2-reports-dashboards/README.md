# 📊 Salesforce Administrator Track - Day 2

## 🎯 Objective

The objective of Day 2 was to understand how Salesforce Reports and Dashboards help organizations analyze data, track performance, monitor KPIs, and support business decision-making.

---

## 📖 What is a Report?

A Report is a list of records that meet specific criteria. Reports help users organize, filter, summarize, and analyze Salesforce data.

### Uses of Reports
- Track business performance
- Analyze data trends
- Monitor records
- Generate business insights
- Support decision-making

---

## 📖 What is a Dashboard?

A Dashboard is a visual representation of report data using charts, graphs, tables, and metrics.

### Uses of Dashboards
- Monitor KPIs
- View business performance at a glance
- Analyze trends visually
- Support management decisions

---

## ⚖️ Difference Between Reports and Dashboards

| Reports | Dashboards |
|----------|------------|
| Display detailed data | Display visual summaries |
| Used for analysis | Used for monitoring |
| Can be exported | Built using reports |
| Show records and data | Show charts and metrics |
| Detailed information | High-level overview |

# 📋 Five Important College Reports

### 1. Student Attendance Report
- Tracks student attendance records
- Viewed by Faculty and Principal
- Refreshed Daily

### 2. Placement Status Report
- Tracks placed and unplaced students
- Viewed by Placement Cell and Management
- Refreshed Weekly

### 3. Faculty Workload Report
- Tracks teaching assignments
- Viewed by HODs and Principal
- Refreshed Weekly

### 4. Internship Tracking Report
- Tracks student internship participation
- Viewed by Placement Team
- Refreshed Weekly

### 5. Student Performance Report
- Tracks academic performance
- Viewed by Faculty and Management
- Refreshed Monthly
